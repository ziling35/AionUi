# LingAI 启动指南

## 环境要求

- **Node.js**: >= 22 < 25
- **包管理器**: bun（推荐）

## 前置检查

```bash
# 检查环境
just preflight
```

## 一、主应用启动（Electron 桌面端）

```bash
cd LingAI

# 安装依赖（首次运行）
bun install

# 开发模式启动
bun run dev
# 或使用 just
just dev
```

## 二、WebUI 启动（浏览器访问）

```bash
cd LingAI

# 开发模式（本地访问）
bun run webui

# 远程访问模式（允许局域网访问）
bun run webui:remote

# 生产模式
bun run webui:prod
bun run webui:prod:remote  # 远程访问
```

启动后访问: http://localhost:3000

## 三、Server 后端服务启动

### 3.1 Admin-API 后端服务

```bash
cd LingAI/server/admin-api

# 安装依赖（首次运行）
bun install

# 初始化数据库（首次运行）
bunx prisma generate
bunx prisma migrate dev

# 启动服务（正确命令）
bunx tsx index.ts
```

服务默认运行在 `http://localhost:3800`

### 3.2 Admin-Web 管理界面（可选）

```bash
cd LingAI/server/admin-web

# 安装依赖
bun install

# 开发模式
bun install
bunx vite
```

## 四、配置文件说明

Server 配置文件位于 `LingAI/server/.env`：

```env
# AI 大模型配置（OpenAI 兼容协议）
OPENAI_BASE_URL=https://api.deepseek.com/v1    # API 地址
OPENAI_API_KEY=your-api-key-here               # API Key
OPENAI_MODEL=deepseek-chat                     # 模型名称

# 服务配置
PORT=3800                                       # 服务端口
HOST=0.0.0.0                                   # 监听地址
```

### 支持的模型平台

| 平台        | BASE_URL 示例                                     |
| ----------- | ------------------------------------------------- |
| DeepSeek    | https://api.deepseek.com/v1                       |
| 通义千问    | https://dashscope.aliyuncs.com/compatible-mode/v1 |
| 智谱 GLM    | https://open.bigmodel.cn/api/paas/v4              |
| 月之暗面    | https://api.moonshot.cn/v1                        |
| 百川        | https://api.baichuan-ai.com/v1                    |
| 本地 Ollama | http://localhost:11434/v1                         |

## 五、常用命令汇总

| 命令                                 | 说明                   |
| ------------------------------------ | ---------------------- |
| `bun run dev`                        | 启动 Electron 桌面应用 |
| `bun run webui`                      | 启动 WebUI（本地）     |
| `bun run webui:remote`               | 启动 WebUI（远程访问） |
| `bunx tsx server/admin-api/index.ts` | 启动 Admin-API 后端    |
| `just preflight`                     | 检查环境依赖           |
| `just install`                       | 安装所有依赖           |
| `just build`                         | 构建生产版本           |
| `just clean`                         | 清理构建产物           |

## 六、常见问题

### 1. 依赖安装失败

```bash
# 清理缓存后重试
bun install --force
```

### 2. 端口被占用

修改 `LingAI/server/.env` 中的 `PORT`，如改为 3801

### 3. Node 版本不匹配

```bash
# 使用 nvm 切换版本
nvm use 22
```

### 4. 数据库未初始化

```bash
cd LingAI/server/admin-api
bunx prisma generate
bunx prisma migrate dev
```
